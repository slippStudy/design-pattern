# 프로젝트 목적
- 19차 SLIPP 스터디를 통해서 디자인 패턴에 대한 학습을 진행함
- 일반적으로 사용하는 설정들에 레퍼런스를 만들어서 다른 프로젝트에도 활용함

# 개발환경
- Java 11
- Spring Boot 2.2.x
- QueryDSL 4.3.x
- MySQL 8.0.x
- Gradle 6.5.x

# Docker Compose를 활용한 로컬 개발환경 구성
- 개발에 필요한 다양한 인프라를 Docker를 이용하여 설정함
- Docker Compose 설정을 통해서 간단하게 환경을 구축할 수 있도록 함
